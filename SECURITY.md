# Security Policy

## Reporting a Vulnerability

Apache Directory follows the [ASF security process](https://www.apache.org/security/). Report privately to
`security@apache.org` (PMC: `private@directory.apache.org`); do not open public issues/PRs for security reports.

## Threat Model

`apache/directory-ldap-api` is the LDAP protocol API and ASN.1/BER codec library within the Apache Directory project. Its security context is covered by the Apache
Directory umbrella threat model (LDAP codec addendum): https://github.com/apache/directory-server/blob/master/THREAT_MODEL.md
